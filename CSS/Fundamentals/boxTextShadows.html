Understanding Shadows:
Box Shadow vs. Text Shadow

Explore how CSS shadows add depth and visual appeal to elements and text.

Box Shadow

The box-shadow property applies a shadow effect to an element's entire box. It can be used to create depth, lift elements off the page, or add decorative effects.

Example Box

Syntax & Effect:

.element-with-shadow {
    box-shadow: offset-x offset-y blur-radius spread-radius color;
    /* Example: */
    box-shadow: 8px 8px 15px 2px rgba(0, 0, 0, 0.4);
}
  • offset-x: Horizontal shadow position.
  • offset-y: Vertical shadow position.
  • blur-radius: Blur effect distance.
  • spread-radius: Shadow size extension.
  • color: Color of the shadow.
  • Can also include inset for an inner shadow.

Text Shadow

The text-shadow property applies a shadow effect to the text content itself. It's useful for enhancing readability or creating artistic typographic effects.

Shadowed Text

Syntax & Effect:

.text-with-shadow {
    text-shadow: offset-x offset-y blur-radius color;
    /* Example: */
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}
  • offset-x: Horizontal shadow position.
  • offset-y: Vertical shadow position.
  • blur-radius: Blur effect distance.
  • color: Color of the shadow.
  • Does not have a spread-radius like box-shadow.
